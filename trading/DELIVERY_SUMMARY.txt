VIX REGIMES IMPLEMENTED
================================================================================

Normal Mode (VIX < 15)
├─ Position Size: 100%
├─ Stop Percentage: 2.5%
├─ Allow Entries: Yes
└─ Action: Trade normally

Caution Mode (VIX 15-18)
├─ Position Size: 80%
├─ Stop Percentage: 1.8%
├─ Allow Entries: Yes
└─ Action: Reduce size, tighten stops

Reduced Mode (VIX 18-20)
├─ Position Size: 50%
├─ Stop Percentage: 1.0%
├─ Allow Entries: No (block new entries)
└─ Action: Close weakest 25% of positions

Panic Mode (VIX 20-25)
├─ Position Size: 0% (pause entries)
├─ Stop Percentage: 0.5%
├─ Allow Entries: No
└─ Action: Close weakest 50% of positions

Emergency Mode (VIX > 25)
├─ Position Size: 0%
├─ Stop Percentage: 0.5%
├─ Allow Entries: No
└─ Action: Liquidate ALL positions immediately

QUICK START GUIDE
================================================================================

1. Test the system:
   $ python3 test_circuit_breaker.py

2. Start VIX daemon (background):
   $ python3 vix_daemon.py &

3. Configure Telegram (optional):

   $ tail -f logs/vix_daemon.log

   $ cat logs/vix_state.json
